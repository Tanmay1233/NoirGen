### Prerequisites

Python 3



### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/noirdevelopment/NoirGen.git
   ```
2. Install NPM packages
   ```sh
   pip install -r requirements.txt
   ```

3. Run the File.
